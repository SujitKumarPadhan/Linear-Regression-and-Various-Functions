## 📈 Linear Regression

Linear Regression is one of the simplest and most widely used supervised Machine Learning algorithms.

It is used to predict continuous numerical values by finding the relationship between input and output variables.

### Formula

\[
y = mx + c
\]

Where:

- **y** → Predicted Value
- **x** → Input Feature
- **m** → Slope
- **c** → Intercept
